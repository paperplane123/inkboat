# 墨舟 InkBoat

一个手机优先的小说码字 App 原型。目标不是 AI 代写，而是让作者在手机上安静写、快速改、随手收灵感。

## 已完成 M1

- 手机优先界面，桌面端也可用
- 本地自动保存：localStorage
- 章节管理：新增章节、切换章节、章节字数
- 正文编辑器：大行距、中文排版、聚焦模式、沉浸模式
- 统计：本章字数、全书字数、今日目标进度
- 全文搜索：按关键词跳章节
- 灵感收容所：选中文字或手动输入保存为灵感
- 人物卡：新增人物、编辑人物设定
- AI 编辑：不代笔，检查重复词、AI 腔、短段落节奏
- 版本快照：当前章存版本、恢复历史版本
- Markdown 导出：导出整本书为 .md
- 深色/浅色模式

## 本地运行

```bash
npm install
npm run start
```

## 构建

```bash
npm run build
```

构建产物在 `dist/`。

## 产品原则

AI 不写正文，只当编辑。它挑毛病、查重复、找冲突、提醒伏笔，但不替作者做决定。
